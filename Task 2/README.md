# Deloitte Data Analytics Virtual Experience – Task 2

## Objective

Analyze employee compensation data to identify potential gender pay inequality across different factories and job roles.

## Task

Using the provided Equality Table dataset, classify each employee group's Equality Score into an Equality Class based on predefined thresholds.

### Classification Rules

* **Fair**: Equality Score between -10 and +10 (inclusive)
* **Unfair**: Equality Score between -20 and -11 or between +11 and +20
* **Highly Discriminative**: Equality Score less than -20 or greater than +20

## Tools Used

* Microsoft Excel

## Outcome

Added a new **Equality Class** column to the dataset and categorized all records according to the specified equality criteria to support further analysis of gender pay equality across the organization.
